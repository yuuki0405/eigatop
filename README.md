# 映画レコメンデーション Web アプリ

## 🔥 概要

この Web アプリは、ユーザが好きな映画を最大3つ選択すると、その映画のジャンル傾向に基づきおすすめ映画を5件表示します。  
選択なしの場合は、全ユーザ評価の高い映画トップ5を表示します。

MovieLens データセット（movies_100k.csv / ratings_100k.csv）を使用し、Python の Flask で実装しています。

---

## 🎯 機能要件

- 3つ以上の映画を選択可能
- 「オススメ映画を表示」ボタン押下で推薦映画画面に遷移
- 選択映画の傾向に基づくレコメンデーション
- 選択がない場合は人気の映画トップ5を表示
- 映画は MovieLens のデータを利用（CSV 参照）

---

## 🧠 採用したレコメンデーションアルゴリズムと理由

### ✅ 映画が選択された場合（選択あり）

**コンテンツベースフィルタリング**

選択された映画のジャンル列を集計し、類似のジャンル映画を絞り込んで評価平均の高いものを推薦します。

**理由**
- 映画同士の内容（ジャンル）から類似性を測る方法が適している
- ユーザの趣味傾向（選択された映画ジャンル）を直接反映できる

---

### ✅ 映画が未選択の場合

**人気順レコメンド**

全ユーザの平均評価が高い映画トップ5を表示します。

**理由**
- ユーザ固有の情報がない場合、映画全体の人気指標を基準にするのが自然
- 実装がシンプルで、結果も妥当性が高い

---

## 🧠 画面遷移とロジック

以下の流れで推薦結果が表示されます。

1. トップ画面で映画を最大3つ選択
2. 「オススメ映画を表示」を押下
3. フロントエンドが POST で選択映画タイトルを送信
4. バックエンド（Flask）で CSV を読み込み
5. 選択状態に応じて処理を分岐
   - 未選択 → 平均評価トップ5
   - 選択あり → ジャンル集計 → 同ジャンル映画を抽出 → 評価の高い上位5
6. 結果をテンプレートに埋め込み
7. ブラウザに推薦映画リストを表示
